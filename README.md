## Overview
This project is a **speech-based AI Sales Assistant** designed to simulate real-world sales conversations. Users can interact with the assistant using voice input. The system converts speech to text, analyzes customer intent, and responds intelligently using text-to-speech. All interactions are stored in a structured JSON format, and a simple UI enables smooth user interaction.

## System Workflow
1. Customer provides voice input  
2. Speech is converted to text  
3. Text is stored in a JSON file  
4. User intent is identified  
5. Appropriate sales response is generated  
6. Response is converted to speech and played back
